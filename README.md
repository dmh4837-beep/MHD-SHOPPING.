# MHD-SHOPPING.
MHD SHOPPING est une plateforme E-commerce moderne développée avec  Next.js, NestJS et postgreSQL. Elle permet la vente en ligne des produits avec gestion des commandes, suivi des livraisons, tableau de bord d'administration, gestion des clients, vendeurs et livreurs. Le projet est optimisé pour mobile, sécurisé et conçus pour être évolutif.
